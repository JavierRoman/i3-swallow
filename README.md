# i3-swallow
used to swallow (send to the scratchpad) a terminal window after a blocking application is run in i3. Requires [i3ipc-python](https://github.com/acrisci/i3ipc-python/).

## usage
i3-swallow [-d] cmd  
 -d Don't return swallowed word on exit  
 cmd command to run
